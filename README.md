## The 30-Second Version

```
Chips = Logic chips + Memory chips + Analog chips
Logic = CPUs (Intel, AMD, ARM) + GPUs (NVIDIA, AMD) + AI chips
Memory = DRAM (Micron, Samsung, SK Hynix) + NAND Flash (Micron, SanDisk, Samsung)
Making chips = Design (fabless) → Manufacturing (foundry: TSMC) → Packaging
The AI boom = massive demand for NVIDIA GPUs + HBM memory from Micron/Samsung
```
